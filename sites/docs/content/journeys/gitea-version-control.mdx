# Learn Gitea Version Control

Outcome: understand how the platform uses Gitea as an internal Git host, not as
a GitHub wrapper: it owns repos, deploy keys, package registry paths, app repo
seeding, and the source that Argo CD reads.

## Run

```bash
cd ~/Developer/personal/platform
make -C kubernetes/kind 500 apply AUTO_APPROVE=1
kubectl -n gitea get deploy,svc,secret
```

At stage `700`, the platform also seeds app repos and enables the local runner:

```bash
make -C kubernetes/kind 700 apply AUTO_APPROVE=1
kubectl -n gitea-runner get deploy,pods
```

## Inspect

Gitea runs as an Argo CD managed application:

```bash
kubectl -n argocd get app gitea -o wide
kubectl -n gitea get svc gitea-http gitea-ssh
kubectl -n gitea get secret gitea-admin-secret
```

Check the repo sync resources:

```bash
rg -n "sync_gitea|app_repo|policies_repo" terraform/kubernetes/gitops.tf
rg -n "gitea_repo_owner|gitea_org" terraform/kubernetes/*.tf
```

The stage files make the learning target explicit:

```hcl filename="kubernetes/kind/stages/700-app-repos.tfvars"
enable_actions_runner      = true
enable_app_repo_subnetcalc = true
enable_app_repo_sentiment  = true
```

## Try One Change

After stage `700`, clone a seeded app repo from local Gitea, make a harmless
documentation change, and inspect the remote:

```bash
git clone ssh://git@127.0.0.1:30022/platform/sentiment.git /tmp/platform-sentiment
cd /tmp/platform-sentiment
git remote -v
git log --oneline -5
```

If SSH access is not active for your runtime, use the runtime's documented
port-forward path from the Lima or Slicer walkthroughs.

<Callout type="warning">
  Gitea is local infrastructure in this platform. Use it to learn GitOps and
  internal package flows without relying on GitHub.
</Callout>

### Acceptance Checks

```bash
curl -s http://127.0.0.1:30090/api/v1/repos/platform/hello-platform
git ls-remote ssh://git@127.0.0.1:30022/platform/hello-platform.git
kubectl -n gitea get pvc
```

Use the PVC check to identify where repository state lives before trying any
backup or restore exercise.
