SD-WAN Headers Explained

When traffic goes through an SD-WAN edge device or tunnel, additional headers are added to preserve original client information:

X-Forwarded-For: Client IP, proxy1, proxy2
Example: 203.0.113.50, 10.0.0.1, 10.0.0.2

X-Real-IP: The actual client IP behind proxies
Example: 203.0.113.50
