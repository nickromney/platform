## Capstone Path

A capstone that would join the journeys together:

```text
Create a tiny "hello-platform" service.
Deploy it to dev through Gitea and Argo CD.
Protect it with Keycloak SSO.
Expose it through Gateway API.
Add Cilium ingress and egress policy.
Add Kyverno-compliant labels and security context.
Observe one request in Hubble and Grafana.
Promote the same image to UAT.
```

<Callout type="warning">
  Run the capstone in a local stack first. The exercise touches routes, GitOps
  state, and policy, so make each change in source and inspect the rendered
  objects before applying the next one.
</Callout>
