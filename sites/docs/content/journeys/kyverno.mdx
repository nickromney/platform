# Learn Kyverno

Outcome: understand how Kyverno admission complements Cilium runtime policy:
Kyverno decides which Kubernetes objects are accepted, generated, or reported;
Cilium decides which traffic is allowed after pods exist.

## Run

```bash
cd ~/Developer/personal/platform
make -C kubernetes/kind 600 apply AUTO_APPROVE=1
kubectl -n kyverno get deploy,pods
kubectl get cpol
```

Validate the checked-in policies:

```bash
scripts/validate-kyverno-policies.sh --mode static --execute
scripts/validate-kyverno-policies.sh --mode live --execute
```

## Inspect

Start with generated default deny:

```bash
kubectl get ns --show-labels | rg "kyverno.io/isolate|namespace-role"
kubectl -n dev get networkpolicy
kubectl -n sit get networkpolicy
```

Then inspect policy reports:

```bash
kubectl get policyreports -A
kubectl get clusterpolicyreports
kubectl get cpol restrict-image-registries -o yaml
```

## Try One Change

Create a local throwaway manifest that is missing the required labels and test
how Kyverno responds:

```yaml filename="bad-deployment.yaml"
apiVersion: apps/v1
kind: Deployment
metadata:
  name: missing-platform-labels
  namespace: dev
spec:
  replicas: 1
  selector:
    matchLabels:
      app: missing-platform-labels
  template:
    metadata:
      labels:
        app: missing-platform-labels
    spec:
      containers:
        - name: pause
          image: registry.k8s.io/pause:3.10
```

```bash
kubectl apply --dry-run=server -f bad-deployment.yaml
```

Read the admission response and map it back to
`require-app-labels-application-namespaces.yaml`.

<Callout type="warning">
  Audit mode is not enforcement. A policy report is evidence that something
  happened; it is not the same as blocking the object at admission time.
</Callout>

### Acceptance Checks

```bash
scripts/validate-kyverno-policies.sh --mode static --execute
kubectl apply --dry-run=server -f unsafe-pod.yaml
kubectl get policyreports -A | rg uat
```

The learning outcome is that a junior engineer can explain why the policy is
in audit mode, what would change to enforce it, and how test fixtures prove the
decision.
